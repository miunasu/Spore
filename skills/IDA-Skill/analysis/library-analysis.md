# 静态库/SDK 分析

本文档提供静态库和第三方 SDK 的识别与分析方法。

## 函数识别

| 目标 | 方法 | 关键 API |
|------|------|----------|
| FLIRT 签名 | 应用签名库 | `ida_libfuncs` |
| 字符串特征 | 找版本/版权字符串 | `ida_strlist` |
| 代码模式 | 识别编译器特征 | `ida_ua.decode_insn()` |

## 第三方库识别

| 库 | 特征 | 识别方法 |
|----|------|----------|
| OpenSSL | "OpenSSL" 字符串 | 字符串搜索 |
| zlib | "inflate" 字符串 | 字符串搜索 |
| SQLite | "sqlite" 字符串 | 字符串搜索 |
| Lua | "lua" 字符串、字节码 | 字符串搜索 |
| V8 | "V8" 字符串 | 字符串搜索 |
